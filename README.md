MegaStore

MegaStore is a full-stack e-commerce platform built with **React.js**, **JavaScript**, **Tailwind CSS**, and **Firebase**. The project is designed to provide a smooth and responsive shopping experience with product browsing, dynamic navigation, authentication, and database integration.

## Features

1.User Side
- Browse product listings
- View product details
- Dynamic routing for different pages
- Responsive design for mobile, tablet, and desktop
- Firebase authentication for user login/signup
- Database integration for storing and fetching product data

2.Admin Panel
- Add new products
- Update product details
- Delete products
- Manage product listings from one place

3.Tech Stack
- React.js – Frontend library for building the user interface
- JavaScript – Core programming language used in the project
- Tailwind CSS – Utility-first CSS framework for responsive styling
- Firebase Authentication – User authentication and login system
- Firebase Database / Firestore – Product data storage and management

4.Project Purpose
This project was built to practice and showcase full-stack web development skills by creating a real-world e-commerce application with authentication, routing, responsive UI, and admin product management.
